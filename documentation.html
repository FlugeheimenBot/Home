Введение

Flugeheimen Bot — это многофункциональный Discord-бот с широким спектром возможностей, включая автоматизацию задач, модерацию и многое другое.

Установка и настройка

Привет друг, если ты читаешь это - значит тебе интересно как установить бота и настроить, я тебе помогу!

Начнём с установки:

Заходишь на наш официальный сайт, листаешь вниз и находишь кнопку"Добавить в дискорд" нажимаешь, выбираешь нужный сервер.

Первичная настройка:

  • /set_bad_words - установи плохие слова - и бот будет за них мутить.
  • /set_welcome_channel - установи канал для приветствий.
  • /set_welcome - установи приветственное сообщение которое будет отправляться в канал который ты установил
  • /set_welcome_dm - установи приветственное сообщение в ЛС и бот будет его отправлять в ЛС.
  • P.S. все приветственные сообщения стоят по умолчанию если их не изменить, также чтобы упомянуть человека в приветственных сообщениях - пропешите {member.mention} Текст приветственного сообщения.

Команды

Модерация

  • /ban [пользователь] [причина(не обязательно)] — Забанить участника (Только для администраторов)
  • /kick [пользователь] [причина(не обязательно)] — Кикнуть участника (Только для администраторов)
  • /mute [пользователь] [причина(не обязательно)] — Замьютить (Только для администраторов)
  • /unmute [пользователь] — Размьютить (Только для администраторов)
  • /add_adm_role [роль] — Добавить роль администратора (Только для администраторов)
  • /unban [пользователь] — Разбанить участника по ID (Только для администраторов)
  • /setup_bad_words — Настроить плохие слова (Только для администраторов)
  • /send_embed — Отправить сообщение в виде Эмбеда (Только для администраторов)
  • /set_auto_role — Настроить автоматическую роль для новых участников (Только для администраторов)

Экономика

  • /balance — Ваш баланс
  • /transfer [пользователь] [сумма] — Перевести монеты

Магазин ролей

  • /add_role_to_shop [роль] — Добавляет роль в магазин (Только для администраторов)
  • /shop — Магазин ролей

Тикеты

  • /create_ticket — Создать тикет
  • /close_ticket — Закрыть тикет

Приветствия

  • /set_welcome — Изменить приветственное сообщение
  • /set_welcome_dm — Изменить приветственное сообщение в ЛС
  • /set_welcome_channel — Установить канал для приветствий

Развлечения

  • /math [выражение] — Сложить или вычесть
  • /create_channel — Создаёт голосовой канал для создания других голосовых каналов (Только для администраторов)
  • /send_embed — Отправить сообщение в виде Эмбеда
